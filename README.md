## ABOUT
- Prototype UI for scanning barcodes and creating inventory
- Usecase is for User to view the React app on their smartphone and add new items into their inventory

## FEATURES:
- Search bar that filters while you type
    - Category filtering
- Each item is displayed as a "card" in a scrollable container
- Modal form for creating new record for inventory
- Uses react-qr-barcode-scanner so that User can scan barcodes with their smartphone camera

## Running locally
- use: npm run dev
- Then access https://YOUR-IP:3000 to view the app on another device
